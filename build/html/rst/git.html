git 儲存庫¶

Git 是一種分散式版本控制系統，主要用於管理程式碼和協作開發。

建立儲存庫¶

在自己的電腦上建立一個 Git 儲存庫。

Pulsar編輯器中按 CTRL+8 即可出現Git畫面.

按下如圖第二步驟出現在畫面再執行第三步驟就完成儲存庫的完成.
